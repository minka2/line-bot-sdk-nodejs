# DatetimePickerAction

Ƭ DatetimePickerAction: Object

When a control associated with this action is tapped, a postback eventopen in new window is returned via webhook with the date and time selected by the user from the date and time selection dialog.

The datetime picker action does not support time zones.

# Date and time format

The date and time formats for the initial, max, and min values are shown below. The full-date, time-hour, and time-minute formats follow the RFC3339open in new window protocol.

ModeFormatExample
datefull-date (Max: 2100-12-31; Min: 1900-01-01)2017-06-18
timetime-hour:time-minute (Max: 23:59; Min: 00:00)00:0006:1523:59
datetimefull-dateTtime-hour:time-minute or full-datettime-hour:time-minute (Max: 2100-12-31T23:59; Min: 1900-01-01T00:00)2017-06-18T06:152017-06-18t06:15
